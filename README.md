# Delivery Box -- A lockable box for delivery prople to deliver stuff to.

This is a design for a delivery box -- a lockable box that delivery drivers
can place deliveries into, using one-time codes, where the delivery can be 
securely  housed  until the resident can retrieve  them.  This is to allow for
secure deliveries when noone is home to receive them.  The box will be
lockable and can be secured.  This will make deliveries secure from 
"porch pirates".

# This is the box design

The box would be made of CDX (exterier grade) 3/8 plywood with internier 2x2
framing, 20 inches by 24 inches by 30 inches, which should be large enough
for  more  deliveries.  It  will  have  wheels  to  make  moving  it  easy  --
partitularly handy for people with long driveways.
